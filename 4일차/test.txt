*git
=>파일 관리
=>파일을 추적(git이 보면서)하여 파일을 기록함

*git hub
=>저장소 (클라우드)
=>git을 사용한 기록을 ui로 표현해줌
=>배포 가능

*집에서 설치
1. git을  구글에 검색해서 설치
2. 깃허브 데스크탑 검색해서 설치
3. 설치한 깃허브 데스크탑 열어서 로그인 및 인증하기
4. 저장소(레포지토리) 만들기
5. create repositories 세번째 클릭
6. 창이 뜨면 레포지토리명 -> 하단에 저장할 경로 -> 레포지토리 생성
7. 생성하면 지정한 경로 폴더안의 폴더를 vscode로 열기
8. 파일 만들고, 코드 작성하던대로 작성하기
9. 깃허브 데스크탑 열어서 좌측하단에 Summary (required)부분에 기록할 타이틀 작성(커밋 과정)
10. 하단 commit to main 버튼 클릭하여 커밋 남기기(기록)
11. 기록을 업로드 할때 상단 탭 메뉴 중 Publish repository 버튼 클릭하여 업로드 시키키
12. 깃허브 페이지에서 새로고침 하여 잘 업로드 되있는지 확인

*파일이 없거나 처음 설치 환경일때
=>깃허브 사이트에서 내려 받고자 하는 레포지토리 클릭
=>중앙 상단 <> code 버튼을 클릭하여 
=>open with github desktop 클릭하여 프로그램 열기
=>경로 실종됐으면 clone으로 깃허브에 있는 파일 복제 하기
